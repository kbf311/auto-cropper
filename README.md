# 自動画像余白切り抜きツール (Auto Cropper)

本ツールは、指定したフォルダ内にある複数のPNG画像から共通の余白（背景色）を自動で検出し、一括で綺麗に切り抜くことができるWindows用ツールです。

---

## 🌟 特徴

*   **自動余白検出**: 画像の四隅から背景色を自動で判定し、余白の大きさを検出します。
*   **一括切り抜き**: フォルダ内のすべての画像を、共通の最小余白で一括処理します。
*   **中心維持モード**: キャラクターなどの位置（画像の中心）をずらさずに余白だけをカットする機能があります。
*   **かんたん起動**: 面倒なコマンド入力は不要で、バッチファイルをダブルクリックするだけで動かせます。

---

## 💻 動作環境

*   **OS**: Windows 10 / 11
*   **必須ソフト**: **Python** (インストールの際、「Add Python to PATH」にチェックを入れている必要があります)

---

## 🛠️ 初回セットアップ方法

ツールを初めて使うときだけ、以下の手順を行ってください。

1.  本ツールのフォルダ（`auto_cropper.py` などがあるフォルダ）を開きます。
2.  `run_setup.bat` をダブルクリックして実行します。
3.  黒い画面が表示され、自動で必要なプログラム（Pillow）がインストールされます。
4.  画面に「セットアップ完了」と表示されたら、キーボードの何かキーを押して画面を閉じます。

これで準備は完了です。

---

## 🚀 使い方

1.  `run_app.bat` をダブルクリックして起動します。
2.  フォルダ選択画面が表示されるので、**処理したいPNG画像が入っているフォルダ**を選択します。
3.  画面に「〇〇個の画像が見つかりました。中心を維持して切り取りますか？」という確認画面が出ます。
    *   **「はい」を選択**: 画像の中心位置を保ったまま切り抜きます（キャラクターの位置などを固定したい場合におすすめ）。
    *   **「いいえ」を選択**: 左右上下の余白をギリギリまで限界までカットします。
4.  処理が完了すると、「処理が完了しました。」とメッセージが表示されます。
5.  選択したフォルダの中に **`convert`** という新しいフォルダが作成され、その中に切り抜かれた画像が保存されます。
    *   ※元の画像は上書きされず、そのまま残ります。
    *   ※処理の詳細はツールと同じフォルダに出力される `auto_cropper.log` というテキストファイルで確認できます。

---

## 💡 2つの切り抜きモードの違い

| モード | メリット | こんなときにおすすめ |
| :--- | :--- | :--- |
| **中心維持モード**<br>（「はい」を選択） | 画像の中心位置がずれません。キャラクターや小物の立ち位置を揃えたまま余白を削ることができます。 | ゲームの立ち絵、アニメーション素材、スプライトシート用の画像など。 |
| **通常モード**<br>（「いいえ」を選択） | 上下左右の余白をそれぞれギリギリまで削るため、最も画像サイズが小さくなります。 | 純粋に余白をなくして画像をコンパクトにしたいとき。 |

---

## ⚠️ 注意事項

*   **対応画像形式**: **PNG画像のみ**対応しています（拡張子が `.png` のもの）。
*   **画像サイズの統一**: 選択したフォルダ内のPNG画像は、**すべて同じ縦横サイズ（ピクセル）**である必要があります。サイズが異なる画像が混ざっているとエラーになり、処理が中断されます。
*   **背景色の自動判定**: 背景色は画像の四隅（左上、右上、左下、右下）のピクセルから自動で判定されます。四隅が透明または単色でない場合、余白が正しく検出されないことがあります。
